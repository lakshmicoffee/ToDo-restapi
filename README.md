 ## Installation Prerequisites
 ### Fake REST API - JSON SERVER
 #### Json Server Installation
 ```sh
npm install -g json-server
```
 #### Serve REST Service
 ```sh
cd fakeRestApi
json-server --watch db.json

## Project Installation
```sh
npm install
npm run serve
```
